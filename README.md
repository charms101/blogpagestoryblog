# 📜 The Storybook Blog

A vintage, story-like blog with an animated night town background, a bird that delivers a scroll, and a secret publisher dashboard for writing and managing posts.

---

## ✨ Features

**For Readers**
- Animated raven flies in and delivers a rolled scroll on page load
- Scroll unrolls itself to reveal the table of contents
- Illustrated night town background with twinkling stars, glowing windows, and flickering candles
- Click any blog entry to read it on a vintage parchment-style page
- Leave comments on any post

**For the Publisher (you)**
- Type `applepb` anywhere on the page to instantly open the Publisher's Study
- Write new blog entries
- Edit existing posts
- Delete posts
- View, edit, and delete reader comments

---

## 🗂️ File Structure

```
storybook-blog/
├── index.html    — page structure and markup
├── style.css     — all styling and animations
└── script.js     — all interactivity and publisher logic
```

---

## 🚀 Running Locally

1. Open the folder in **VS Code**
2. Install the **Live Server** extension (by Ritwick Dey)
3. Open `index.html` and click **Go Live** in the bottom right
4. Opens in your browser at `localhost:5500`

> Do not just double-click `index.html` — the browser will block the CSS and JS from loading. Always use Live Server.

---

## 🔐 Publisher Mode

Type a secret password anywhere on the page.

The Publisher's Study opens instantly with:
- **Entries tab** — write new posts, edit or delete existing ones
- **Comments tab** — read, edit, or delete reader comments

To exit, click **✕ Exit Study** in the top right of the dashboard.
